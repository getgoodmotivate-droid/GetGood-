**On Computer:**
- **Windows:** Press `Ctrl + Shift + R`
- **Mac:** Press `Cmd + Shift + R`
- **Chrome/Edge:** Press `Ctrl + F5`

**On Mobile:**
- **Android Chrome:**
  1. Menu (3 dots)
  2. Settings → Privacy → Clear browsing data
  3. Select "Cached images and files"
  4. Clear
  5. Reload page

- **iPhone Safari:**
  1. Settings app → Safari
  2. Clear History and Website Data
  3. Reopen Safari
  4. Visit app URL
